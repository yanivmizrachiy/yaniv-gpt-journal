# GENERATOR_SPEC

השאלה נוצרת לפי קטגוריות, אחוזים ומספר ניסיונות.
התשובה מחושבת אוטומטית.

## רמות קושי
- בסיסית: שאלה ישירה של חישוב מספר מופעים צפוי.
- בינונית: אותה מיומנות, בתוספת דרישה להציג חישוב קצר.
- מתקדמת: אותה מיומנות, בתוספת ייצוג ההסתברות כשבר פשוט והסבר מדוע מדובר בתוצאה צפויה בערך ולא בתוצאה ודאית.
